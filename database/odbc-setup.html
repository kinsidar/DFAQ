Configure MS SQL ODBC Connection

Configure Microsoft SQL Database ODBC Connection

  1. Locate and run ODBC Data Source Administrator with the appropriate Windows version

  2. Select the appropriate tab to add ODBC driver to (e.g. User DSN for specific user)

  1. Click "Add" and select the appropriate ODBC driver

  1. Fill out name, description, server (server will be the one created through this guide)

  1. Select authenticity verification otion the same as the connecting server

  1. Select your database

  1. Click finish

  1. Test data source, and you are done!
